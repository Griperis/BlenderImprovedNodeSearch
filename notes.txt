Node Search

What user problem this solves?
- you have a clusterfuck of nodes and you want to filter out nodes to get a grasp about what is happening
- you want to get overview of attributes
- you want to get information about node flow


Functionality:
- search by type
- search by socket
- search by name
    - search by regex
    - search by prefix
    - search by kw match
- search node groups
    - search where node group is used
- search missing
    - search missing images nodes
    - search non-connected nodes
    - search missing node groups
- search unconnected nodes
- search empty attributes
- search attributes

UI
- list number of found nodes
- allow iterating over them
- focus found node and highlight where it is
